# キネマ旬報 （日本《电影旬报〉）

几次评选：

- 1995 年，电影诞生百年，评选了最佳 100 部日本片，共有 107 部；外国（非日本）电影【缺】
- 1999 年，【电影旬报】创刊 80 周年，评选出 20 世纪百佳日本电影，共有 119 部；外国（非日本）电影 129 部。
- 2009 年，【电影旬报】创刊 90 周年，评选出“映画遺産 200”，其中日本电影共 198 部，外国（非日本）电影 225 部。
