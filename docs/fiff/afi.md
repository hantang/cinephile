# AFI 美国电影协会 Top100

AFI 百年百大电影

[美国电影协会评选的美国影史百部佳片](https://www.douban.com/doulist/207326/)
[AFI百年百大电影 - douban.com](https://www.douban.com/doulist/10926253/)

> AFI百年百大电影的入选与排名根据如下条件判断：
>（1）电影至少要有40分钟长。
>（2）电影必须是英语电影，
>（3）并且重要的创作元素或资金来源必须是来自美国。
>（4）电影必须在电影界或各大电影节获过奖。
>（5）电影要通过技术创新或有远见的叙事方法或其他突破性成就对事情进行记录。
